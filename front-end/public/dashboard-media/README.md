# Dashboard Media

Drop your frontend artwork here.

- Hero image: `hero/hero-main.png`
- Product images:
  - `products/white-cap.png`
  - `products/grey-tee.png`
  - `products/denim-shirt.png`
  - `products/white-sneaker.png`

Current matching rules:

- names containing `hat`, `cap`, `horee` -> `white-cap.png`
- names containing `denim`, `emporia`, `armoni`, `button` -> `denim-shirt.png`
- names containing `v-neck`, `tshirt`, `t-shirt`, `tee` -> `grey-tee.png`
- names containing `shoe`, `sneaker`, `adisia`, `speed` -> `white-sneaker.png`

If a file is missing, the UI will show a styled placeholder instead of a broken image.
